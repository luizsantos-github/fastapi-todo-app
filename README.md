# Todo App

Simple todo app that uses FastAPI features

# 📌 Overview

Part of my hands on learning for Fast API

# 🚀 Features
- CRUD API operations for a simple TODO app

# 🛠 Tech Stack

- Language / Framework: Python
- Library / Tools: Uvicorn, FastAPI, SQLite

# 📂 Project Structure

```
fastapi-todo/ 
│── db/ 
│── services/ 
│── main.py 
│── README.md 
│── requirements.txt 
```

# ⚙️ Installation
Clone repository

`git clone https://github.com/luizsantos-github/fastapi-todo-app.git`

Enter folder: `cd fastapi-todo`

Create Virtual Environment: `python3 -m venv .venv`

Install dependencies: `pip install -r requirements.txt`

Run the project: (Uvicorn) `uvicorn main:app --reload`
